# Test Case – Login with Valid Credentials

**ID:** TC-002  
**Title:** Verify login with valid username and password  
**Preconditions:**  
- User already registered in the system with valid credentials.  
- Browser opened on the login page.  

**Steps:**  
1. Enter a valid username.  
2. Enter the correct password.  
3. Click on "Login".  

**Expected Result:**  
- The system must grant access to the user’s account.  
- The homepage (or dashboard) should be displayed.
